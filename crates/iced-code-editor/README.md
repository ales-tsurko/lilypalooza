# iced-code-editor

This crate is a vendored fork of the upstream `iced-code-editor` project:

- Upstream repository: <https://github.com/LuDog71FR/iced-code-editor>

It is kept in-tree for `lily-view` so we can integrate and customize the editor without using a git submodule or depending on upstream release timing.
